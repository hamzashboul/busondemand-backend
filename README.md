# BusOnDemand — Backend

## Description
Smart bus booking system for Isra University students.

## Tech Stack
- Node.js + Express
- MySQL
- Socket.io
- JWT Authentication

## Installation
npm install

## Run Development
npm run dev

## Run Production
npm start

## Environment Variables
PORT=3000
DB_HOST=your_db_host
DB_USER=your_db_user
DB_PASSWORD=your_db_password
DB_NAME=busondemand
JWT_SECRET=your_jwt_secret
FRONTEND_URL=your_frontend_url
